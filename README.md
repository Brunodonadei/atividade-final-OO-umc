# Backend Brechó - Versão OO

Estrutura orientada a objetos com classes para Models, Repositories, Services e Controllers.

## Como rodar

1. npm install
2. npm start

Endpoints:
- POST /produtos
- GET /produtos
- GET /produtos/:id
- PUT /produtos/:id
- DELETE /produtos/:id

- POST /reservas
- GET /reservas
- DELETE /reservas/:id
